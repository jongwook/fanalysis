1)
For each $$V_i$$, we can take a sequence of Hilbertian basis set $$E_i = \left \{e_{1} , \cdots , e_{d_i} \right \}$$ so that $$V_i = \mathrm{span}( E_i )$$, by incrementally taking the orthonormal basis set for each $$V_i$$. Note that $$d_i$$ denotes the dimension of $$V_i$$.

Then $$E = \cup_i E_i$$ is a Hilbertian basis of $$H$$, since $$\overline{\mathrm{span}(E)} = \overline{\cup_i V_i} = H$$. Let $$d$$ denote the dimension of $$H$$, which can be finite or infinite, and we have $$d_i \rightarrow d$$ as $$i \rightarrow \infty$$.

So for any $$x \in H$$ and all $$i$$, $$P_i (x) = \sum_{k=0}^{d} \left < e_k , x \right > e_k $$. We know that by Corollary 1 of week 5 that $$ \sum_{k = d_i + 1}^{d} \| \left < e_k , x  \right > \|^2 \rightarrow 0 $$ as $$ i \rightarrow \infty$$, so we have

$$ \left \| x - P_i (x) \right \|^2 = \left \| \sum_{k = d_i + 1}^d \left < e_k , x \right > e_k \right \|^2 = \sum_{k = d_i + 1}^d \left \| \left < e_k , x \right > \right \| ^2 \rightarrow 0 $$

as $$i \rightarrow \infty$$. Therefore, $$P_i (x) \rightarrow x$$ as $$ i \rightarrow \infty$$.


2)
Since $$H$$ is Hilbert, weak convergence and weak* convergence is equivalent. Using the definition of weak* convergence, for all $$i$$, 

$$\lim_{n \rightarrow \infty} P_i ( x_n ) \\
= \lim_{n \rightarrow \infty} \sum_{k = 0}^{d_i} \left < e_k, x_n \right > e_k \\
= \sum_{k = 0}^{d_i} \left ( \lim_{n \rightarrow \infty}  \left < e_k, x_n \right > \right ) e_k \\
= \sum_{k = 0}^{d_i} \left < e_k, x \right > e_k \\
= \phantom{\int} P_i (x)$$
