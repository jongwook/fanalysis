$$d$$ is a distance function since:
  • For all $$(x, y) \in E \times E$$, $$d(x, y) \ge 0$$, because the number of letters that differ is obviously zero or more.
  • For all $$(x, y) \in E \times E$$, $$d(x, y) = 0 \Leftrightarrow x = y$$,
    since if the number of letters that differ is zero then two words have to be the same, 
    and if the two words are the same then the number of letters that differ are zero.
  • For all $$(x, y) \in E \times E$$, $$d(x, y) = d(y, x)$$,
    since the number of letters that differ between two words are the same regardless of the order of two words,
  • For all $$(x, y, z) \in E \times E \times E$$, $$d(x, y) \le d(x, z) + d(z, y)$$,
    because if $$d(x, z) + d(z, y) < d(x, y)$$, then we can alter $$d(x, z) + d(z, y)$$ letters to change $$x$$ from $$y$$ by going from $$x$$ to $$z$$ and then $$y$$, which contradicts $$d(x, z) + d(z, y) < d(x, y)$$.
