$$T$$ is a topology since:
  • Both the empty set and $$\mathbb{R}$$ are open.
    $$\emptyset$$ is open since the scope ($$x \in O$$) of the universal quantifier is empty. (vacuously true)
    $$X = \mathbb{R}$$ is open since for all $$x \in \mathbb{R}$$ and $$\epsilon > 0$$, $$]x-\epsilon, x+\epsilon[ \; \subset \mathbb{R}$$ holds.

  • Any union of open sets is open.
    Let $$O_1, O_2, O_3, \cdots$$ be open sets in $$T$$, and let $$O=\bigcup_i O_i$$.
    By definition of union, for any $$x \in O$$ there exists $$i \in \mathbb{N}$$ such that $$x \in O_i$$, 
    and since $$O_i \subset T$$, there exists $$\epsilon$$ such that $$]x - \epsilon, x + \epsilon[ \; \subset O_i$$. 
    Since $$O_i \subset O$$ for all $$i$$, we just showed that $$\forall x \in O \; \exists \epsilon > 0 :\; ]x-\epsilon , x + \epsilon [ \; \subset O$$.

  • Any intersection of finitely many open sets is open.
    Let $$O_1, O_2, O_3, \cdots, O_n$$ be open sets in $$T$$ and $$O=\bigcap_i O_i$$. 
    Let x be any element of $$O$$. By definition of intersection, $$x \in O_i$$ for all $$i \in \{ 1, 2, \cdots, n \}$$,
    so for all $$i \in \{ 1, 2, \cdots, n \}$$ there exists $$\epsilon_i > 0$$ such that $$]x - \epsilon_i, x + \epsilon_i[\; \subset O_i$$.
    Let $$\epsilon = \min _i \epsilon_i > 0$$, then $$]x - \epsilon, x + \epsilon[ \subset O_i$$ holds for all $$i$$, thus $$]x - \epsilon, x + \epsilon[\; \subset O$$
    we just showed that $$\forall x \in O \; \exists \epsilon > 0 :\; ]x-\epsilon , x + \epsilon [ \; \subset O$$.
